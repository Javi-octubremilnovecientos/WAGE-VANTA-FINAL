This is the **Wage Vantage** project — an international salary comparison SaaS application. The app helps users compare salaries across countries, roles, and experience levels using statistical visualizations.

Built with React, Redux Toolkit, RTK Query, Recharts, Tailwind CSS v4, and Supabase (REST API).

The project uses specialized agents (in `.github/agents/`) for different domains:
- Redux architecture (@redux-architect)
- Data visualization (@recharts-expert)
- UI/UX with Tailwind (@ui-ux-master)
- React Router navigation (@navigation-architect)
- Supabase backend (@supabase-expert)
- Git/CI/CD (@git-github-expert)
